About - Credit Card Fraud Detection

Cette documentation fournit un aperçu du script Python pour construire et évaluer un modèle de détection de fraude en utilisant un ensemble de classificateurs d'apprentissage automatique. Le script lit les données des clients à partir d'un fichier CSV, les prétraite, entraîne un modèle d'ensemble et évalue ses performances.

  • Chargement et prétraitement des données

Avant d'exécuter le code, assurez-vous d'avoir les bibliothèques suivantes installées :

  • pandas
  • numpy

  • matplotlib
  • plotly.express
  • plotly.figure_factory
  • scikit-learn

Chargement et prétraitement des données

Le script charge les données des clients à partir d'un fichier CSV nommé "Customer_Data.csv" en utilisant la bibliothèque Pandas.
Le prétraitement des données comprend la gestion des valeurs manquantes, le calcul des scores Z pour la détection de fraude et le marquage des enregistrements potentiellement frauduleux.

Visualisation des données

Le script génère des histogrammes pour chaque colonne du jeu de données, colorés par la colonne 'FRAUD_FLAG', afin de visualiser les distributions des données.

Construction et entraînement du modèle

Le script construit un modèle d'ensemble en utilisant trois classificateurs : la régression logistique, la forêt aléatoire et AdaBoost.
Il utilise OneHotEncoder et ColumnTransformer pour le prétraitement des caractéristiques.
Le modèle d'ensemble est construit en utilisant un VotingClassifier avec un vote 'soft'.

Évaluation du modèle

Le jeu de données est divisé en ensembles d'entraînement et de test. Le modèle est entraîné sur les données d'entraînement et évalué en utilisant les métriques suivantes :

  • Précision
  • Matrice de confusion
  • Rapport de classification

Courbe ROC et AUC

Le script calcule la courbe ROC (Receiver Operating Characteristic) et la surface sous la courbe ROC (AUC) pour évaluer les performances du modèle à distinguer les cas de fraude des non-fraude.

Sérialisation du modèle

Le modèle d'ensemble entraîné est sérialisé en utilisant la bibliothèque pickle et enregistré dans un fichier nommé "ensemble_model.pkl".

Utilisation

Pour utiliser ce code :

Assurez-vous d'avoir les bibliothèques requises installées comme mentionné dans la section "Prérequis".
Chargez vos données clients dans un fichier CSV nommé "Customer_Data.csv".
Exécutez le script pour prétraiter les données, entraîner le modèle et évaluer ses performances.
Le modèle sérialisé peut être chargé et utilisé pour faire des prédictions.

Conclusion

Ce script Python fournit une solution complète de détection de fraude en utilisant des techniques d'ensemble d'apprentissage automatique. Il peut servir de point de départ pour développer un système de détection de fraude pour votre cas d'utilisation spécifique. Assurez-vous de le personnaliser davantage pour répondre à vos besoins et à vos données.
